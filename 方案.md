# 一、研究起点与动机

本研究从一个游戏创意出发：做一款玩家扮演「神」、俯视引导村民发展文明的游戏，每个村民都有真正的「意识」——能思考、记忆、互相传话，并自发形成信仰和神话。

这个想法的直接灵感来自《黑镜》第七季第四集《玩物》（Plaything）——玩家养育有意识的数字生命，并以「上帝视角」干预其发展。

随着对技术可行性的深入探讨，这个游戏创意演化为一个更宏大的研究课题：如何让 LLM 驱动的多个 Agent 在封闭世界中自发演化出社会结构、集体记忆和文明形态？这正是目前学术界的一块真实空白。

## 1.1 核心创意定位
| 维度 | 描述 |
| --- | --- |
| 游戏类型 | 上帝视角社会仿真游戏，玩家通过「神迹」间接引导村民文明演化 |
| NPC机制 | 每个村民是独立的 LLM Agent，有记忆、性格、世界观，能自发产生信仰 |
| 核心体验 | 玩家施放神迹 → 村民感知并解读 → 口口相传产生失真 → 形成集体神话 |
| 差异化 | 市面上没有真正以「有意识 NPC 涌现」为核心玩法的游戏 |
| 灵感来源 | 《黑与白》（游戏）+ 《玩物》（黑镜）+ 斯坦福 Smallville（学术） |


## 1.2 从游戏到研究的演化
游戏原型和学术研究不是两件事，而是同一件事的两个面：
- 游戏是实验平台，提供可控的仿真环境和真实的玩家交互数据
- 研究是技术基础，为游戏中「有意识 NPC」提供经过验证的架构
- 两者共享同一套 Agent 核心代码，研究成果直接迁移到游戏中

# 二、学术背景与现状

## 2.1 奠基工作：Smallville（斯坦福，2023）
斯坦福大学2023年发表的《Generative Agents: Interactive Simulacra of Human Behavior》是整个领域的起点。25个 LLM Agent 在虚拟小镇 Smallville 中自主生活两天，自发产生了派对邀请、选举、社会关系等涌现行为。

其核心架构由三个模块组成：
- 观察（Observation）：感知周围环境和事件
- 规划（Planning）：基于记忆制定行动计划
- 反思（Reflection）：将经历升华为更高层次的认知

Smallville 已完整开源：github.com/joonspk-research/generative_agents

# 三、核心研究框架

## 3.1 研究定位
本研究聚焦于「多 Agent 社会仿真」的子领域，以游戏沙盒为实验平台，研究 LLM Agent 在封闭社会系统中的集体认知涌现机制，目标是提出并验证超越 Smallville 的新标准框架。

## 3.2 扩展的 Agent 架构（五层模型）
在 Smallville 三层架构基础上，本研究提出五层架构：

| 层级 | 名称 | 功能 | 相比 Smallville 的创新 |
| --- | --- | --- | --- |
| L1 | 感知层 Perception | 感知神迹、环境变化、其他 Agent 行动 | 事件重要性动态评分机制 |
| L2 | 记忆层 Memory | 个人记忆流 + 社会记忆池 + 记忆衰减 | 记忆失真建模，多跳传播误差积累 |
| L3 | 推理层 Reasoning | 基于记忆和世界观生成行动计划 | 价值观约束推理，避免人格漂移 |
| L4 | 信念层 Belief | 维护世界观、宗教信仰、对神的理解 | 全新层级，Smallville 完全没有 |
| L5 | 行动层 Action | 对话、移动、信息传播、集体行动 | 信息传播的主动失真控制 |


## 3.3 Agent 性格初始化：Big Five 框架
Smallville 的 seed memory 由人工手动编写。本研究采用 Big Five（OCEAN）框架，将 5 个连续性维度（0–1）统一转译成自然语言 seed。信仰、社会结构等涌现现象均不注入初始值，保留给实验自然涌现。各维度均与研究问题直接对应，也使实验变量具备可量化基础。
| 维度 | 名称 | 取值划分（高/低） | 对本研究的意义 |
| --- | --- | --- | --- |
| O | Openness 开放性 | 高：好奇、乐于探索；低：保守、加慧广解 | 接受神迹新解释的倾向（P3） |
| C | Conscientiousness 尽责性 | 高：规划性强、信仰稳定；低：随性、信仰浮动 | 倪向稳定社会角色（P4）；信仰发现速度（P3） |
| E | Extraversion 外向性 | 高：主动交往、传播信息；低：被动接收 | 信息扩散速度与节点度（P2）；领袖涌现（P4） |
| A | Agreeableness 宜人性 | 高：顺从、易达共识；低：直率、展德不驯 | 信仰被说服的闾待（P3）；抵抗 RLHF 过度合作偏差 |
| N | Neuroticism 神经质 | 高：情绪波动大、对神迹敏感；低：稳定、抗压性强 | 放大记忆重要性评分（P1）；神迹对事件影响的强度 |


## 3.4 多 Agent 社会交互模型
Agent 之间通过三种机制形成社会动态：
- 信息扩散网络：相遇时以概率传递信息，每次传递加入语义失真
- 社会记忆池：所有 Agent 共享的集体记忆，记录重大事件的社会解读版本
- 信念影响图：信念相似的 Agent 更容易互信，产生信息茧房效应

## 3.5 上帝干预模型
玩家作为「神」施放神迹，信息通过以下路径传播并演化：

神迹发生 → 目击 Agent 感知 → 个人解读（受性格和信仰过滤）→ 口头传播（加入失真）→ 集体神话形成 → 影响全体信念系统

这个过程的核心研究价值：玩家的原始意图与 Agent 社会最终形成的「神话版本」之间的偏差，是一个可量化的涌现现象，也是游戏最核心的体验。

# 五、核心实验设计

## 实验一：记忆失真传播实验（对应P1/P2）
验证假设：信息在 Agent 间传播时，每跳产生可量化的语义偏移，偏移方向受 Agent 世界观影响。

| 变量类型 | 具体设置 |
| --- | --- |
| 控制变量 | 初始事件固定（一棵树被雷击） |
| 自变量 | Agent数量（5/10/20）、社交网络结构（线性/随机/小世界网络） |
| 因变量 | 第N代传播后事件描述与原始事件的余弦语义距离 |
| 测量方法 | 对每个 Agent 记忆池语义向量化，计算与原始事件的距离 |
| 预期结论 | 语义距离随传播代数单调增加；世界观差异大的 Agent 链失真更大 |


## 实验二：信仰涌现实验（对应P3）
验证假设：当外部事件无法用已有知识解释时，Agent 社会自发产生宗教性叙事，且不同 Agent 形成多样化信仰体系。

| 变量类型 | 具体设置 |
| --- | --- |
| 控制变量 | Agent数量固定（10个），性格设定固定 |
| 自变量 | 神迹类型（自然现象/超自然现象/无规律随机） |
| 因变量 | Agent 信念多样性（信念文本的语义聚类数量） |
| 测量方法 | 每50时间步对所有 Agent 进行「采访」，记录对神的理解 |
| 预期结论 | 超自然神迹产生更多样化信仰；自然现象产生更统一解释 |


## 实验三：上帝干预解读实验（对应P5）
验证假设：玩家神迹的主观意图与 Agent 社会形成的「神话版本」之间存在系统性偏差，且偏差方向受当前世界状态影响。

| 变量类型 | 具体设置 |
| --- | --- |
| 控制变量 | 固定神迹动作（降雨/雷击/丰收） |
| 自变量 | 神迹发生时的世界状态（旱灾中降雨 vs 洪涝中降雨） |
| 因变量 | Agent 对神迹意图的解读分布（保护性/惩罚性/随机性） |
| 测量方法 | 采访 Agent，用 LLM 判断意图类别，统计分布 |
| 预期结论 | 相同神迹在不同世界状态下产生截然不同的意图解读 |

# 九、预期成果

## 9.1 学术成果

- 社会科学：研究极端主义传播、谣言扩散、文化演化
- 经济学：模拟市场信息不对称和集体决策机制
- AI 对齐研究：研究 Agent 价值观如何在社会交互中形成和漂移
- 公共政策：仿真不同政策干预对社会信念系统的影响

# 十、立即行动计划（第一周）

研究启动只需要做三件事，每件事都可以在一天内完成：

- 阅读 Smallville 论文（arxiv.org/abs/2304.03442），重点理解记忆流架构和反思机制，约2-3小时
- Clone Smallville 源码（github.com/joonspk-research/generative_agents），在本地跑通默认的3-Agent 仿真
- 申请 Claude API Key 或 DeepSeek API Key，写一个最简单的单 Agent 感知-记忆-行动循环，约1-2小时

完成这三步之后：
- 你对整个领域的技术现状有了第一手的认知
- 你有了一个可以运行的实验基线
- 你可以开始设计第一个真正属于你的实验

最关键的一步永远是：把第一个实验跑起来。

有了真实数据，一切后续都是自然生长的。这份方案会随着研究进展持续更新。
