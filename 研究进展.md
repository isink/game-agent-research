## 一、已完成事项

### 1.1 研究方案梳理

完整阅读 `方案.docx`（v2.0），核心框架：

- **研究目标**：用 LLM 驱动多 Agent 社会仿真，研究封闭世界中集体记忆、信仰涌现、社会结构自发形成
- **双轨并行**：学术研究（目标 ICLR/ACL/AAAI）+ 上帝视角游戏（Godot 4）
- **五层架构**：感知 → 记忆 → 推理 → 信念（新增）→ 行动
- **六大切入点**：P1 记忆系统 → P2 信息传播 → P3 信仰涌现 → P4 社会结构 → P5 干预效应 → P6 文明演化

### 1.2 Smallville 论文精读

**论文**：Generative Agents: Interactive Simulacra of Human Behavior（斯坦福，2023）  
**arXiv**：2304.03442

#### 核心架构（三层）

| 层级 | 功能 | 关键实现 |
|------|------|---------|
| Memory Stream | 记忆流，按时间追加的自然语言列表 | 三维打分检索：recency × importance × relevance |
| Reflection | 反思，触发阈值150，每天2-3次 | 两步：提炼问题 → 检索记忆 → 生成洞察 → 写回记忆流 |
| Planning | 规划，top-down 递归分解 | 全天计划 → 小时级 → 15分钟级 |

#### 重要性打分 Prompt（可直接复用）

```
On the scale of 1 to 10, where 1 is purely mundane (e.g., brushing teeth)
and 10 is extremely poignant (e.g., a break up, college acceptance),
rate the likely poignancy of the following piece of memory.
Memory: [...]
Rating: <fill in>
```

#### 论文承认的局限 → 对应本研究机会

| 局限 | 对应切入点 |
|------|-----------|
| 记忆检索失败，Agent 会编造记忆 | P1：记忆失真建模 |
| 信息原样传递，无失真机制 | P2：谣言/失真传播 |
| 无信念/世界观层 | P3：信仰涌现（L4 信念层） |
| 无权力结构，Agent 平等 | P4：社会结构涌现 |
| 只跑了两天游戏时间 | P6：长时演化 |
| RLHF 导致 Agent 过度合作 | P3 实验需对抗此偏差 |

#### 评估方法（可借用）

「采访」Agent 五类问题：自我认知 / 记忆 / 规划 / 反应 / 反思  
消融实验：去掉任意一层（记忆/反思/规划），可信度均显著下降

### 1.3 Seed Memory 设计讨论

**结论**：Seed memory 不必完全人工编写，采用 **Big Five + 涌现** 的混合方案：

| 内容 | 来源 |
|------|------|
| 名字、年龄、职业 | Seed 注入 |
| 性格倾向（Big Five 五维度） | Seed 注入（参数化生成自然语言） |
| 世界观、信仰、对神的理解 | **完全涌现，不注入** |
| 对神迹的解读 | **完全涌现，不注入** |

### 1.4 Big Five 框架引入

已更新 `方案.docx`，新增 **3.3 Agent 性格初始化：Big Five 框架**。

五个维度与研究问题的对应：

| 维度 | 对本研究的意义 |
|------|--------------|
| O 开放性 | 接受神迹新解释的倾向（P3） |
| C 尽责性 | 倾向稳定社会角色（P4）；信仰发现速度（P3） |
| E 外向性 | 信息扩散速度与节点度（P2）；领袖涌现（P4） |
| A 宜人性 | 信仰被说服的阈值（P3）；抵抗 RLHF 过度合作偏差 |
| N 神经质 | 放大记忆重要性评分（P1）；神迹对事件影响的强度 |

**实验价值**：固定其他四维，只改变单一维度，可得到干净的实验变量（审稿人友好）。

### 1.5 Smallville 源码阅读

**路径**：`GameAgent/generative_agents-main/reverie/backend_server/`

#### ConceptNode 结构（associative_memory.py:19）

```python
self.type         # "event" / "thought" / "chat"
self.depth        # 反思层级：event=0，thought=1+
self.subject/predicate/object  # 主谓宾结构
self.poignancy    # 重要性分数（1-10）
self.last_accessed  # 用于 recency 计算
self.expiration   # 有字段但从未被实际使用
```

#### 打分公式真实实现（retrieve.py:244）

**论文写的**：α_recency = α_importance = α_relevance = 1  
**代码实际**：
```python
gw = [0.5, 3, 2]  # recency, relevance, importance
score = recency_w * recency * 0.5
      + relevance_w * relevance * 3
      + importance_w * importance * 2
```
作者注释：*"in the future, these weights should likely be learned, perhaps through an RL-like process"*  
→ **P1 直接切入点：根据 Big Five N 维度动态调整 importance 权重**

#### Recency 实现缺陷（retrieve.py:145）

```python
recency_vals = [recency_decay ** i for i in range(1, len(nodes) + 1)]
```
按**列表位置**衰减，不是按**真实时间差**衰减。  
→ Agent 长时间不活动时，旧记忆排名异常升高  
→ **P1 改进点：改为按真实时间差计算衰减**

#### 信息传播无失真（associative_memory.py:317）

```python
# 完全基于关键词精确匹配，无语义失真
for i in contents:
    if i in self.kw_to_event:
        ret += self.kw_to_event[i.lower()]
```
→ **P2 改进点：每次 Agent 间传播加入语义失真层**

## 三、关键设计决策记录

| 决策 | 结论 | 理由 |
|------|------|------|
| 性格框架选型 | Big Five，不用 MBTI | 心理学标准工具，信效度有保证，审稿人可接受 |
| Seed memory 方式 | Big Five 参数化生成自然语言 | 可量化、可控制实验变量、保留涌现空间 |
| 信仰/世界观初始化 | 完全不注入，靠涌现 | 研究目标就是观察信仰如何自发形成 |
| 验证架构 LLM | DeepSeek V3 | 极低成本，跑通架构阶段够用 |
| 正式实验 LLM | Claude Haiku | 效果稳定，结果可引用 |
